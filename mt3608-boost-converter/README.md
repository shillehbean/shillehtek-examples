# MT3608 Boost Converter

The MT3608 is an adjustable DC-DC step-up (boost) converter used to raise a lower-voltage source (like a LiPo or battery pack) to a higher regulated output. Developers use it to power 5V/9V/12V peripherals from single-cell LiPo, AA packs, or other low-voltage supplies and to size input power sources appropriately for their load.
