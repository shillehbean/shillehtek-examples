# 4x4 Matrix Keypad

A 4x4 membrane matrix keypad is a 16-key input arranged in a 4x4 grid for simple user input. Developers can use it to add numeric entry, PIN pads, menu navigation, or basic control inputs to microcontroller projects; included examples show usage with Arduino and MicroPython.
