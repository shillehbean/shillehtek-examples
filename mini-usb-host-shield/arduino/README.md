# Arduino examples

- [`example_1.ino`](./example_1.ino) — Initializes the MAX3421E-based USB host, reports initialization status over Serial, and runs the USB task loop while prompting to plug in a device.
- [`example_2.ino`](./example_2.ino) — Sets up the USB host as a HID keyboard reader and echoes characters from a connected USB keyboard to the Serial console.
- [`example_3.ino`](./example_3.ino) — Uses the PS4USB driver to read input from a wired PS4 controller over USB and prints analog stick positions and a button press state to Serial when connected.
