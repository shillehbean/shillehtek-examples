# ESP8266 D1 Mini V3

The ESP8266 D1 Mini V3 is a compact ESP8266-based Wi‑Fi development board suitable for small IoT projects. Developers can use it to build Wi‑Fi sensors, simple web servers, and remote data loggers using either the Arduino core or MicroPython.
