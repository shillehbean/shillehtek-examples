# Arduino examples

- [`example_1.ino`](./example_1.ino) — Connects the D1 Mini to a Wi‑Fi network and prints its IP address and RSSI to the serial console.
- [`example_2.ino`](./example_2.ino) — Reads the board's single ADC channel (A0) and prints the raw 10-bit value and the converted 0–3.2V voltage to serial.
