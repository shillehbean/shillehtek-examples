# TXS0108E

The TXS0108E is an eight-channel, high-speed, full-duplex bi-directional logic level converter for interfacing 3.3V hosts with 5V peripherals. Developers can use it to connect I2C, UART, PWM, and general-purpose signals between a 3.3V microcontroller and 5V devices — for example, reading 5V I2C sensors from a Pi, driving a 5V LED strip from a Pico, or bridging serial between an ESP32 and a 5V device.
