# Arduino examples

- [`example_1.ino`](./example_1.ino) — Creates a bidirectional serial passthrough so you can type AT commands to the ESP-01 from the Arduino Serial Monitor and see responses.
- [`example_2.ino`](./example_2.ino) — Sends AT commands to set the ESP-01 to station mode and join a specified Wi‑Fi network, then queries and prints the module's IP address.
- [`example_3.ino`](./example_3.ino) — Uses AT commands to open a TCP connection to example.com and perform a simple HTTP GET request, printing the server response.
