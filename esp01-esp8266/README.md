# ESP-01 ESP8266

The ESP-01 is a compact ESP8266-based serial Wi‑Fi transceiver used to add wireless connectivity to microcontroller projects. Developers can use it with an Arduino or USB-serial adapter to send AT commands, join Wi‑Fi networks, and perform HTTP requests from constrained hosts.
