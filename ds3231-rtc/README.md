# DS3231 RTC Module

The DS3231 RTC is a precision I2C real-time clock with an onboard temperature sensor and coin-cell backup for battery-backed timekeeping. Developers use it to add accurate, persistent time to projects such as data loggers, alarm clocks, or scheduled controllers on microcontrollers and single-board computers.
