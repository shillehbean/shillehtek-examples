# 650nm Red Laser Module

The 650nm Red Laser is a compact 5mW diode laser that produces a visible red beam useful for alignment, pointers, and simple optical experiments. Developers can build projects such as a blinking indicator, a laser tripwire using an LDR, or a Pi-controlled on/off switch. The examples show safe switching via an NPN transistor and basic beam-break detection.
