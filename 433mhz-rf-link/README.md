# 433MHz RF Link Kit

The 433MHz RF Link Kit is a basic ASK/OOK transmitter and receiver pair for low-speed wireless communication. Developers can use it to build remote controls, simple wireless sensors, or data links between microcontrollers and a Raspberry Pi.
