# Arduino examples

- [`example_1.ino`](./example_1.ino) — Sends a repeating "Hello, world!" message over a 433 MHz ASK transmitter using the RadioHead RH_ASK driver on Arduino (TX on digital pin 12).
- [`example_2.ino`](./example_2.ino) — Receives messages from a 433 MHz ASK receiver using the RadioHead RH_ASK driver on Arduino (RX on digital pin 11) and prints received text to the serial console.
