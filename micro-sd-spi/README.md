# Micro SD SPI Module

This is a Micro SD / TF card adapter with an SPI interface for microcontrollers. It provides removable storage for projects such as data loggers, file readers, and configuration storage. The examples show basic read/write usage on Arduino, ESP32, and MicroPython (RP2040) platforms.
