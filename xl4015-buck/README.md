# XL4015 Buck Module

The XL4015 Buck is an adjustable DC-DC step-down regulator capable of delivering up to 5 A of output current and handling input voltages up to ~38 V. Developers can use it to build regulated power rails, battery chargers, or voltage-monitoring systems and integrate it with microcontrollers or external ADCs to log or control the output.
