# Arduino examples

- [`example_1.ino`](./example_1.ino) — A simple transmitter sketch that initializes the nRF24L01+, opens a writing pipe, and sends a "Hello World" packet once per second while printing transmission status to Serial.
- [`example_2.ino`](./example_2.ino) — A basic receiver sketch that initializes the nRF24L01+, opens a reading pipe, listens for incoming packets, and prints received text to Serial when available.
