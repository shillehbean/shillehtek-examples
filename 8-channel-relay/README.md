# 8-Channel Relay Module

This is an 8-channel 5V relay board that provides eight opto-isolated active-LOW inputs to switch external loads. Developers can use it to build projects such as home automation, lamp/motor controllers, or other low-cost switched-power projects with microcontrollers or single-board computers.
