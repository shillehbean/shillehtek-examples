# Raspberry Pi examples

- [`example.py`](./example.py) — Uses RPi.GPIO with BCM pins 17, 27, 22, 23, 24, 25, 5, 6 set HIGH (relay off) and loops to turn each relay briefly on (LOW) then off with 0.5 s delays.
