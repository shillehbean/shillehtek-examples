# 8-Channel 12V Relay Module

An 8-channel 12V relay board that provides eight digital inputs (active LOW) for switching external 12V loads from a microcontroller or single-board computer. Developers can use it to build projects such as home automation switches, controlled lighting, or relay-driven motor/pump control using Arduino, ESP32, Raspberry Pi, or MicroPython-capable boards.
