# Arduino examples

- [`example_1.ino`](./example_1.ino) — Read the module's analog SIG on A0, print the raw ADC value at ~50 Hz, and flag a "flex" when the reading exceeds a configurable threshold.
- [`example_2.ino`](./example_2.ino) — Perform a moving-average smoothing over N analog samples from A0 to stabilize the EMG signal and detect flexes more reliably than single-sample thresholding.
