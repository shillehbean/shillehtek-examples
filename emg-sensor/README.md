# EMG Sensor Module

The EMG Sensor reads small electrical signals from muscle activity (electromyography) via a single analog SIG output. Developers can use it to detect muscle flexes and build gesture controls, prosthetic interfaces, or biofeedback projects with microcontrollers or Raspberry Pi plus an ADC.
