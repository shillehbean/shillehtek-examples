# XIAO ESP32-C6

The XIAO ESP32-C6 is a compact Seeed XIAO-form factor development board based on the ESP32-C6 SoC with Wi‑Fi (802.11ax/n/g/b) and BLE LE support. Developers can use it to build low-power Wi‑Fi clients, BLE peripherals, and small IoT sensors or controllers. This folder contains quick-start examples in the ESP32 Arduino core and MicroPython to get hardware and radio features running.
