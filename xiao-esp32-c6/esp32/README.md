# Esp32 examples

- [`example_1.ino`](./example_1.ino) — Blinks an external LED connected to GPIO0 (D0) at a 1 Hz rate using the ESP32 Arduino core.
- [`example_2.ino`](./example_2.ino) — Connects the board to a Wi‑Fi network using the provided SSID/password and prints the IP address, RSSI and free heap to Serial.
- [`example_3.ino`](./example_3.ino) — Initializes NimBLE and starts BLE advertising with the device name 'ShillehTek-C6'.
