# MAX7219 8x8 Module

The MAX7219 8x8 is an SPI-driven LED matrix driver board that controls an 8×8 grid of LEDs. Developers can use it to display characters, icons, and simple animations from microcontrollers such as Arduino or Raspberry Pi Pico. This repo contains example sketches and MicroPython snippets to get you started wiring and programming the display.
