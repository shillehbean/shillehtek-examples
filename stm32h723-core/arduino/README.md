# Arduino examples

- [`example_1.ino`](./example_1.ino) — Blink a LED connected to PB0 (or the built-in LED if wired to PB0) with a 500 ms on/off period using the Arduino-style API on the STM32 board.
- [`example_2.ino`](./example_2.ino) — Send the string "Hello from STM32H723!" over USART1 once per second using STM32 HAL initialization generated by STM32CubeMX.
- [`example_3.ino`](./example_3.ino) — PlatformIO environment configuration for building Arduino-framework firmware for the STM32H723ZGT6 (genericSTM32H723ZG) and uploading/debugging with ST-Link, with a serial monitor at 115200 baud.
