# Raspberry Pi examples

- [`example.py`](./example.py) — Runs on a Raspberry Pi, reads NMEA sentences from /dev/serial0 using pyserial and pynmea2, and prints position, speed, and UTC time when an RMC sentence reports a valid fix.
