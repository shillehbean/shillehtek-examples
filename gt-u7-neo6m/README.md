# GT-U7 NEO-6M GPS

The GT-U7 NEO-6M is a small GPS receiver module that outputs standard NMEA sentences over UART. Developers can use it to build GPS data loggers, basic trackers, or add location/time fixes to microcontroller and single-board computer projects.
