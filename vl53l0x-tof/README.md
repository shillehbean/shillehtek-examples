# VL53L0X

The VL53L0X is a laser time-of-flight (ToF) distance sensor that reports millimeter-accurate range measurements. These examples show how to use the module from Arduino, ESP32 (MicroPython), Raspberry Pi (CircuitPython), and Raspberry Pi Pico (MicroPython) to build rangefinders, proximity sensors, or obstacle-detection projects.
