# SX1262 ESP32-S3

The SX1262 ESP32-S3 is a development board that pairs an ESP32-S3 MCU with an SX1262 LoRa radio and a 0.96" OLED. Developers can use it to build LoRa IoT endpoints, sensor nodes, or connected devices that display status locally on the onboard OLED.
