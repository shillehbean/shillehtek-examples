# Arduino examples

- [`example_1.ino`](./example_1.ino) — Instructions to add the Heltec ESP32 boards URL to Arduino IDE, install the Heltec board package, select the WiFi LoRa 32(V3) board and install the Heltec_ESP32 library.
- [`example_2.ino`](./example_2.ino) — Initializes the OLED display on the Heltec WiFi LoRa 32 V3, powers the Vext rail, and writes a three-line "Hello" message to the screen.
- [`example_3.ino`](./example_3.ino) — Initializes the SX1262 LoRa radio using RadioLib on Heltec V3 pin mapping and demonstrates sending a transmit packet and waiting for a received packet with a 5s timeout.
