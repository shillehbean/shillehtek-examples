# PCA9685

The PCA9685 is a 16-channel, 12-bit PWM/servo driver controlled over I2C. Developers use it to add many servos or LEDs to projects such as multi-servo robotics, LED dimming/lighting rigs, or motor controllers without using many MCU PWM pins.
