# Arduino examples

- [`example_1.ino`](./example_1.ino) — Reads the joystick's X and Y analog axes and the push switch, then prints raw values and switch state to the Serial monitor at 9600 baud.
- [`example_2.ino`](./example_2.ino) — Reads analog axes, applies a center deadzone, converts the analog position into discrete directions (UP/DOWN/LEFT/RIGHT/CENTER), and prints the result to Serial.
