# XIAO nRF52840

The XIAO nRF52840 is a compact microcontroller board based on the Nordic nRF52840 with USB-C and BLE support. This repo contains small examples showing how to cycle the on-board RGB LED, advertise a BLE Battery service, read a 12-bit analog input, and run a CircuitPython blink demo.
