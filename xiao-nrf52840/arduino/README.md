# Arduino examples

- [`example_1.ino`](./example_1.ino) — Sequentially cycles the on-board RGB LED (active LOW) so each color (red, green, blue) turns on for 400 ms.
- [`example_2.ino`](./example_2.ino) — Creates a BLE peripheral advertising the standard Battery Service and updates/notifies a battery level characteristic every 5 seconds.
- [`example_3.ino`](./example_3.ino) — Reads a 12-bit ADC value from A0 and prints the raw sample and its converted voltage to the serial console.
