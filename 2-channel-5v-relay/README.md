# 2-Channel 5V Relay

This is a simple 2-channel 5V relay board that lets a 5V microcontroller or single-board computer switch higher-voltage or higher-current loads. Developers can use it to build projects such as simple home automation switches, timed power control, or remote-controlled lights and motors.
