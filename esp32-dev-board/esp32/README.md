# Esp32 examples

- [`example_1.ino`](./example_1.ino) — Connects the ESP32 to a specified Wi-Fi network and prints the board's local IP address and Wi‑Fi RSSI to the serial console.
- [`example_2.ino`](./example_2.ino) — Initializes the ESP32 as a BLE peripheral and starts advertising with the device name "ShillehTek-ESP32".
