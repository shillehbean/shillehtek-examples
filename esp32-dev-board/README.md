# ESP32 Dev Board

The ESP32 Dev Board is a compact development board based on the ESP-WROOM-32 module with a CP2102 USB-C serial interface and 4 MB flash. Developers can use it to build Wi-Fi and Bluetooth projects such as networked sensors, BLE peripherals, and I2C-connected devices; this repo includes examples for connecting to Wi-Fi, advertising over BLE, and scanning an I2C bus with MicroPython.
