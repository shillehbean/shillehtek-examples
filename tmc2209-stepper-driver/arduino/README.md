# Arduino examples

- [`example_1.ino`](./example_1.ino) — Basic Step/Dir test that enables the driver and rotates a stepper one full revolution forward and then back using STEP and DIR pins.
- [`example_2.ino`](./example_2.ino) — Initializes and configures a TMC2209 via SoftwareSerial using the TMCStepper library (sets current, microsteps, and StealthChop) then steps the motor.
