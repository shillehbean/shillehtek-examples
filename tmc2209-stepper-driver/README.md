# MKS TMC2209

The MKS TMC2209 is a stepper motor driver board with an integrated heatsink and support for Step/Dir and UART control. Developers can use it to drive NEMA stepper motors quietly (StealthChop), configure current and microstepping over UART, and build projects like 3D printer axes, CNC stages, or robotic joints.
