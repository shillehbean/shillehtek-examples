# Arduino examples

- [`example_1.ino`](./example_1.ino) — Enables NeoPixel power via a GPIO and cycles the on-board WS2812 RGB LED through red, green, and blue using the Adafruit NeoPixel library.
- [`example_2.ino`](./example_2.ino) — Reads a 12-bit ADC value from A0 (GP26), converts the raw reading to volts (0–3.3V), and prints both the raw value and voltage to the serial console.
