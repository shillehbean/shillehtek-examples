# XIAO RP2040

The Seeed XIAO RP2040 is a tiny RP2040-based development board with a USB-C connector, a single onboard NeoPixel, and analog-capable pins. Developers use it to build compact projects such as LED indicators, sensor nodes, or small controllers.
