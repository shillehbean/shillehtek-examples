# Joystick Shield V1.A

The Joystick Shield V1.A is an Arduino R3-compatible shield that provides two analog joysticks with push buttons and a servo header for building handheld controllers or input interfaces. Developers can use it to read X/Y axes and button presses for gamepad-style inputs, map joystick motion to servos, or convert analog positions into discrete directions for robots and UI navigation.
