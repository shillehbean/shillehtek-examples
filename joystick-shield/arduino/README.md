# Arduino examples

- [`example_1.ino`](./example_1.ino) — Reads both left and right joystick X/Y axes plus their push buttons and prints the raw values and button states to the serial monitor.
- [`example_2.ino`](./example_2.ino) — Maps the right joystick's X analog value (0–1023) to a 0–180° servo angle and drives a servo connected to digital pin 9.
- [`example_3.ino`](./example_3.ino) — Converts the left joystick's analog X/Y into a discrete direction (LEFT/RIGHT/UP/DOWN/CENTER) using a deadzone and prints the result to serial.
