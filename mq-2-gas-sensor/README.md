# MQ-2 Sensor Module

The MQ-2 is a gas and smoke sensor that provides an analog output (AO) proportional to gas concentration and a digital output (DO) that trips at a configurable threshold. Developers can use it to build simple gas/smoke detectors, audible alarms, or logging/monitoring projects with microcontrollers like Arduino, ESP32, or Raspberry Pi Pico.
