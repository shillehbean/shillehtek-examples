# 24V Relay Module

This is a single-channel 24V relay board that lets a microcontroller or SBC switch a 24V circuit using an active-LOW control input. Developers can use it to build simple power control, automation, or switching projects (for example turning on/off lights, motors, or other 24V loads) while keeping logic and load grounds common.
