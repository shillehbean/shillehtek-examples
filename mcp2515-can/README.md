# MCP2515 CAN Bus Module

The MCP2515 CAN Bus board combines an MCP2515 SPI CAN controller with a TJA1050 transceiver so microcontrollers without native CAN can exchange CAN frames. Developers use it to build vehicle diagnostics tools, CAN data loggers, or networked sensor/control nodes that speak CAN.
