# RC522 RFID Module

The RC522 is a 13.56 MHz SPI-based RFID reader/writer module that works with MIFARE cards and tags. Developers can use it to build access control, asset tagging, or identification projects with Arduino, Raspberry Pi, or MicroPython-capable boards.
