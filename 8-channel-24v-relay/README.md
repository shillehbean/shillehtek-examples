# 8-Channel 24V Relay

An 8-channel 24V relay board with active-LOW control inputs for switching external 24V loads from a microcontroller or single-board computer. Developers can use it to build control projects such as lighting banks, motor/valve switching, or automated test rigs. This repository contains simple wiring and example code for Arduino, ESP32, Raspberry Pi, and MicroPython targets.
