# AC220V Time Delay Relay

This AC220V Time Delay Relay is a single-channel programmable relay with a TTL trigger input for controlling AC mains loads with adjustable delay timing. Developers can connect the module's trigger input to a microcontroller or single-board computer GPIO to build timed control projects (for example delayed lights, fans, or automation sequences) using the examples below.
