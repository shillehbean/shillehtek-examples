# Esp32 examples

- [`example_1.ino`](./example_1.ino) — Connects the ESP32-C3 to a Wi‑Fi network, prints the assigned IP address and RSSI to the serial console, and periodically reports whether the connection is still active.
- [`example_2.ino`](./example_2.ino) — Reads a 10k potentiometer on ADC A0 (GPIO0) with 12-bit resolution, converts the raw value to voltage, and prints both raw and voltage readings to the serial console.
