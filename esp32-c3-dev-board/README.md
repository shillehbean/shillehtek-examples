# ESP32-C3 Dev Board

The ESP32-C3 Dev Board is a compact development board based on Espressif's ESP32-C3 SoC providing Wi‑Fi, Bluetooth LE, ADC, and I2C. Developers can use it to build networked sensors, BLE peripherals, and general embedded projects (for example: Wi‑Fi clients, ADC readings from potentiometers, or I2C device scanners).
