# Arduino examples

- [`example_1.ino`](./example_1.ino) — Basic Arduino sketch that initializes the IRremote library and prints received IR command codes (hex) over Serial.
- [`example_2.ino`](./example_2.ino) — Arduino example that reacts to specific remote button hex codes to control an LED (turn on/off) after you learn the button codes.
