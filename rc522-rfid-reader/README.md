# RC522 RFID Module

The RC522 is a 13.56 MHz RFID reader/writer based on the MFRC522 chip. Developers can use it to build projects that read RFID tags and cards such as access control, asset tracking, or simple identification systems using Arduino, Raspberry Pi, or MicroPython boards.
