# Arduino examples

- [`example_1.ino`](./example_1.ino) — Reads key presses from the 4x3 keypad and prints each pressed key to the serial console.
- [`example_2.ino`](./example_2.ino) — Implements a simple 4-digit PIN entry system using the keypad, with '*' to clear, '#' to submit, and masked feedback via serial indicating ACCESS GRANTED or DENIED.
