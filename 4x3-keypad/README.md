# 4x3 Keypad

This is a 4x3 membrane matrix keypad (four rows by three columns) for reading numeric and symbol input. Developers can use it to add PIN entry, menu navigation, or simple user-input interfaces to microcontroller projects such as door locks, access panels, or control menus.
