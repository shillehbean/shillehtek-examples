# Arduino examples

- [`example_1.ino`](./example_1.ino) — Blink the on-board LED on PC13 (active LOW) using the Arduino STM32 core.
- [`example_2.ino`](./example_2.ino) — Read a 10k potentiometer on PA0 with a 12-bit ADC and print raw value and computed voltage over Serial.
- [`example_3.ino`](./example_3.ino) — Scan the I2C1 bus (SDA=PB7, SCL=PB6) using the Wire library and print detected device addresses.
- [`example_4.ino`](./example_4.ino) — STM32CubeIDE HAL example that toggles the PC13 LED using HAL_GPIO_WritePin and HAL_Delay in a main loop.
