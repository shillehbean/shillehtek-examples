# Blue Pill STM32F103C8T6

The Blue Pill STM32F103C8T6 is a compact ARM Cortex-M3 development board (STM32F103) for embedded projects. Developers can use it to build basic MCU projects such as LED blinkers, analog sensor readers, and I2C device interfaces using the Arduino STM32 core or STM32CubeIDE.
