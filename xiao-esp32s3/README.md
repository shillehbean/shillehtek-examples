# XIAO ESP32S3

The XIAO ESP32S3 is a compact development board built around the Espressif ESP32-S3 microcontroller with pre-soldered pins and a USB-C connector. Developers can use it to build Wi‑Fi connected projects, capacitive-touch interfaces, and small control applications; the examples here show basic LED blink, touch sensing, a tiny HTTP server (ESP32 Arduino core), and a MicroPython blink.
