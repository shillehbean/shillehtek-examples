# Esp32 examples

- [`example_1.ino`](./example_1.ino) — Blinks an external LED connected to D0 (GPIO1) on the XIAO ESP32S3 with a 250 ms on/off cycle using the ESP32 Arduino core.
- [`example_2.ino`](./example_2.ino) — Reads the capacitive touch pad on D2 (TOUCH3), prints the raw touch value to serial, and lights the D0 LED when a touch is detected (value < 20000).
- [`example_3.ino`](./example_3.ino) — Connects the XIAO ESP32S3 to Wi‑Fi and runs a minimal HTTP server that serves a status page showing uptime and free heap memory.
