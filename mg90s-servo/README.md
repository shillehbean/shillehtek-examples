# MG90S Servo

The MG90S is a 9g metal-gear micro servo that provides approximately 180° of rotation and accepts standard hobby servo PWM (50 Hz). Developers can use it for small robots, pan/tilt camera mounts, RC control surfaces, or any project requiring precise angular positioning; this repo includes example code for Arduino, ESP32, Raspberry Pi, and MicroPython.
