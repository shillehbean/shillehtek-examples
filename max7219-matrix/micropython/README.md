# Micropython examples

- [`example.py`](./example.py) — Runs on a Raspberry Pi Pico with the micropython-max7219 driver to scroll a text message across four cascaded 8x8 matrices via SPI (DIN->GP19, CS->GP17, CLK->GP18).
