# Esp32 examples

- [`example.ino`](./example.ino) — Runs on an ESP32 using the micropython-max7219 driver to scroll a message across four cascaded 8x8 matrices via SPI with the specified GPIO pins (DIN->GPIO23, CS->GPIO5, CLK->GPIO18).
