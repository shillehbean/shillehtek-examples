# MAX7219 Matrix

The ShillehTek MAX7219 Matrix is a 4-in-1 LED dot-matrix board composed of four cascaded 8x8 red LED modules driven by a MAX7219. Developers can use it to display scrolling text, simple animations, and numeric readouts from microcontrollers or single-board computers over SPI.
