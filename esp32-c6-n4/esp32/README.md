# Esp32 examples

- [`example_1.ino`](./example_1.ino) — Connect the ESP32-C6 to a 2.4 GHz Wi‑Fi network and print the assigned IP address to the Serial console.
- [`example_2.ino`](./example_2.ino) — Perform a BLE scan using the NimBLE stack and list discovered devices' addresses and RSSI values.
