# ESP32-C6-N4

The ESP32-C6-N4 is a compact development board based on Espressif's ESP32-C6 chip, providing 2.4 GHz Wi‑Fi and BLE connectivity for embedded projects. Developers can use it to build networked sensors, BLE beacons, and simple IoT controllers using Arduino-style or ESP32 toolchains.
