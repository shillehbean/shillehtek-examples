# Arduino examples

- [`example_1.ino`](./example_1.ino) — Cycle the onboard WS2812 RGB LED on GPIO8 through red, green, and blue colors using the Adafruit NeoPixel library.
- [`example_2.ino`](./example_2.ino) — Scan the I2C bus (default SDA=GPIO6, SCL=GPIO7) and print any detected device addresses to Serial.
