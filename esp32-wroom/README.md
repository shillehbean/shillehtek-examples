# ESP32 Dev Board

This repository documents the ESP32 WROOM development board with an onboard CP2102 USB-C serial adapter. Developers can use it to build Wi‑Fi and BLE connected projects, scan I2C sensors, and prototype embedded applications with Arduino/ESP32 or MicroPython code.
