# Esp32 examples

- [`example.ino`](./example.ino) — Outputs a continuous ~1 kHz sine wave from the ESP32 DAC on GPIO25 (DAC1) via a 1 µF coupling capacitor to the TPA3118's Audio IN+ by sending voltage samples at ~8 kHz.

Full wiring + setup notes: [manual](https://shillehtek.com/blogs/shillehtek-product-manuals/tpa3118-pbtl-single-digital-amplifier-board-1x60w-manual)
