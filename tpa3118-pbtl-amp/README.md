# TPA3118 PBTL Amp

This board is a single-channel TPA3118 PBTL class-D digital power amplifier (up to ~60 W) for driving a single speaker. Developers can use it to build mono audio projects, run test tones, or connect microcontroller DACs or PWM outputs to drive a speaker.
