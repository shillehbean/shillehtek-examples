# 24MHz USB Logic Analyzer

This is an 8-channel 24 MHz USB logic analyzer (FX2-based Saleae-compatible clone) for capturing and decoding digital signals. Developers use it to inspect buses like I2C, SPI, UART and to debug microcontroller and embedded hardware by recording waveforms with Saleae Logic 1.x or sigrok/PulseView.
