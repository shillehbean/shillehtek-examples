# Micropython examples

- [`example.py`](./example.py) — Runs on a Raspberry Pi Pico under MicroPython to configure the ADXL345 over I2C, read 16-bit axis data, convert to signed values and m/s², and print the X/Y/Z acceleration repeatedly.
