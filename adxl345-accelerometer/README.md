# ADXL345

The ADXL345 is a small, pre-soldered 3-axis digital accelerometer that reports acceleration and tilt over I2C. Developers can use it to build tilt sensors, motion-detection projects, or simple data-logging and orientation systems on Arduino, Raspberry Pi, Pico, or ESP32 boards.
