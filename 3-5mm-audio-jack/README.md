# 3.5mm Audio Jack Breakout

A small breakout board exposing a pre-soldered 3.5mm stereo audio jack (tip/ring/sleeve) plus a mechanical plug-detect switch. Developers can use it for headphone/speaker output, headset mic input, or plug-detect features in microcontroller projects such as tone generation, audio-level sensing, and insertion detection.
