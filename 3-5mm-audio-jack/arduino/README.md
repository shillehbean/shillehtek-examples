# Arduino examples

- [`example_1.ino`](./example_1.ino) — Read the jack's mechanical detect switch and print "Plug INSERTED" or "Plug REMOVED" over Serial at 9600 baud.
- [`example_2.ino`](./example_2.ino) — Sample an electret microphone AC-coupled through the TRRS jack, compute a simple RMS-like level from multiple analog reads, and print the audio level over Serial.
