# Arduino examples

- [`example_1.ino`](./example_1.ino) — Basic SoftwareSerial echo: forwards text from a paired phone to the Arduino Serial Monitor and echos the same text back to the phone at 9600 baud.
- [`example_2.ino`](./example_2.ino) — AT-command passthrough: puts the HC-05 into AT mode (requires the KEY/EN pin high) and forwards AT commands between the Arduino Serial Monitor and the module at 38400 baud.
