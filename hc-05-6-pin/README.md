# HC-05 6-Pin

The HC-05 is a classic Bluetooth SPP serial transceiver available in a 6-pin footprint for easy wiring to microcontrollers. Developers use it to add wireless serial links to projects like remote sensors, phone-controlled robots, or a wireless console for debugging and control.
