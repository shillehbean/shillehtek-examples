# FT232RL USB‑TTL

The FT232RL USB-TTL adapter is a USB-to-UART bridge used to connect, program, and debug 3.3V/5V TTL serial devices. Typical uses include uploading sketches to an Arduino Pro Mini, flashing ESP-01 modules, and opening serial consoles to capture device output.
