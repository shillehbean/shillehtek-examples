# XIAO ESP32C3

The XIAO ESP32C3 is a compact ESP32-C3-based development board in the XIAO form factor with USB-C connectivity, Wi‑Fi and Bluetooth Low Energy. Developers can use it to build small IoT nodes, BLE beacons, Wi‑Fi utilities, and sensor interfaces; example projects include LED blink, Wi‑Fi scanning, I2C device discovery, and BLE advertising with MicroPython.
