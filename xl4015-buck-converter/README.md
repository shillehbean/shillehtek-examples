# XL4015 Buck Converter

The XL4015 is an adjustable DC‑DC step‑down (buck) converter capable of around 5 A / 75 W and includes a digital voltmeter display for output monitoring. These examples show how to measure and log the module's output voltage with an Arduino or ESP32 so you can build regulated bench supplies, battery chargers, or in‑system power monitors.
