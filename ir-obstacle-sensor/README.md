# IR Obstacle Sensor

The IR Obstacle Sensor is a small infrared reflection sensor that provides a digital output when an object is detected nearby. Developers can use it to add obstacle detection to robots, stop motors when something is close, or trigger actions on microcontrollers and single-board computers.
