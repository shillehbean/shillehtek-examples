# Arduino examples

- [`example_1.ino`](./example_1.ino) — Reads the sensor on digital pin D2 and prints "Obstacle detected!" or "Path clear" to the Serial monitor at 9600 baud.
- [`example_2.ino`](./example_2.ino) — Uses the sensor on D2 to stop or run two motors by PWM-ing motor enable pins (9 and 10), halting the robot when an obstacle is detected.
