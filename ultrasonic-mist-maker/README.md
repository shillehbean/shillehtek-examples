# Ultrasonic Mist Maker

The Ultrasonic Mist Maker is a DC 5V ultrasonic humidifier transducer that produces a fine, cool mist when driven. Developers can control it from microcontrollers or a Raspberry Pi to build small humidifiers, misting effects, or automated plant-watering systems using a MOSFET or a relay.
