# 2-Channel 24V Relay Module

The 2-Channel 24V Relay Module provides two relays controlled by logic-level inputs with active-LOW activation. Developers can use it to switch external 24V loads (lights, solenoids, small motors) from microcontrollers; the examples show basic on/off cycling with Arduino, ESP32, Raspberry Pi, and Raspberry Pi Pico (MicroPython).
