# Nano V3.0 CH340C

The Nano V3.0 CH340C is a compact ATmega328P (16 MHz) development board with a CH340C USB-to-serial chip and USB-C connector. Developers use it to prototype microcontroller projects such as sensor readouts, I2C peripherals, PWM LED control, and USB serial communication. This repo contains simple example sketches to get started with the board's basic I/O and interfaces.
