# Arduino examples

- [`example_1.ino`](./example_1.ino) — Blink the onboard LED on D13 (LED_BUILTIN) at a 1 Hz rate.
- [`example_2.ino`](./example_2.ino) — Send an incrementing counter to the USB serial monitor at 9600 baud.
- [`example_3.ino`](./example_3.ino) — Scan the I2C bus (SDA=A4, SCL=A5) and print any detected device addresses to Serial.
- [`example_4.ino`](./example_4.ino) — Read a 10-bit analog value from A0, convert it to voltage (0–5V), and print raw and voltage readings over Serial.
- [`example_5.ino`](./example_5.ino) — Create a fading (breathing) effect by PWMing an LED on pin D9 with analogWrite.
