# 433MHz Relay

A 433 MHz 12V 1-channel relay with a wireless remote that lets you switch a single load wirelessly. Developers can use an Arduino or Raspberry Pi to emulate the remote, learn the remote's code, or control the relay programmatically in automation projects.
