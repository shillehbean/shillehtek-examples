# Arduino examples

- [`example_1.ino`](./example_1.ino) — Transmit a 433 MHz code from an Arduino using a TX module and the rc-switch library to emulate the relay's wireless remote button.
- [`example_2.ino`](./example_2.ino) — Use an Arduino with a 433 MHz RX module and rc-switch to receive and print the remote's numeric code and bit length for learning the remote signal.
