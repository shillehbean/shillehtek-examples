# Esp32 examples

- [`example_1.ino`](./example_1.ino) — Blink the onboard LED (GPIO2) on the D1 R32 at 500 ms intervals using the Arduino/ESP32 core.
- [`example_2.ino`](./example_2.ino) — Connect the D1 R32 to a Wi‑Fi network using the WiFi.h library and print the assigned IP address to Serial.
