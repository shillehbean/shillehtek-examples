# ESP32 UNO D1 R32

The ESP32 UNO D1 R32 is a CH340G-based ESP32 development board in an Arduino UNO-like form factor. Developers can use it to build Wi-Fi and Bluetooth projects, read analog sensors via the UNO-style headers, and prototype sketches or MicroPython apps.
