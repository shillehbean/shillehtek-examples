# Arduino examples

- [`example_1.ino`](./example_1.ino) — Initializes I2C on GPIO4/GPIO5, starts the SSD1306 128x64 OLED at address 0x3C, and displays a large "Hello!" message.
- [`example_2.ino`](./example_2.ino) — Initializes I2C and the SSD1306 OLED, connects the board to a Wi‑Fi network, then displays connection status and the device IP address on the OLED.
