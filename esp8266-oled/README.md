# ESP8266 OLED

This is a NodeMCU ESP8266 development board with a built-in 0.96" SSD1306 OLED display. Developers can use it to build small IoT status panels, Wi‑Fi connected information displays, or simple sensor/readout projects that show text or graphics on the OLED.
