# 4-Channel 24V Relay Module

This is a 4-channel 24V relay board with active-low inputs that lets a microcontroller switch up to four separate 24V loads (lights, motors, solenoids, etc.) using simple GPIO signals. The examples show how to drive the relays from Arduino, ESP32, Raspberry Pi, and MicroPython boards so you can build home automation, robotics, or basic industrial control projects.
