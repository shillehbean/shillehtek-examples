# Arduino examples

- [`example_1.ino`](./example_1.ino) — Plays alternating 1 kHz and 2 kHz tones on a passive buzzer attached to digital pin 8, with a short silent pause between cycles.
- [`example_2.ino`](./example_2.ino) — Plays a short melody by sending specific note frequencies and durations to the buzzer using tone() in setup().
