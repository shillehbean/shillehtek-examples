# KY-006 Buzzer

The KY-006 is a passive piezo buzzer breakout that produces tones when driven by a microcontroller or any PWM-capable GPIO. Developers can use it to add beeps, alarms, and simple melodies to Arduino, Raspberry Pi Pico, or Raspberry Pi projects.
