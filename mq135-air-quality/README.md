# MQ-135

The MQ-135 is an analog air-quality / hazardous-gas sensor that provides an analog output (AO) and a digital threshold alarm output (DO). Developers can use it to monitor ambient gas levels, trigger alarms, log air-quality data, or control ventilation using Arduino, MicroPython boards, or a Raspberry Pi with an ADC.
