# Micropython examples

- [`example.py`](./example.py) — Runs on a Raspberry Pi Pico with MicroPython: reads AO via ADC0 (accounting for a 2:1 voltage divider), reads DO on GP15, and prints the raw ADC value, reconstructed sensor voltage, and DO state repeatedly.
