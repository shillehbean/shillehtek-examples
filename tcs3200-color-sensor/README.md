# GY-31 TCS3200

The GY-31 TCS3200 is an RGB color sensor breakout that outputs a square-wave frequency proportional to the intensity of reflected red, green, and blue light. Developers can use it to read raw color channel frequencies, normalize values, and build projects like color detectors, sorters, or simple object color recognition systems.
