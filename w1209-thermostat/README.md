# W1209

The W1209 is a compact 12V digital temperature controller/thermostat with an integrated relay and basic setpoint/hysteresis configuration. Developers can use it to build simple temperature-controlled projects such as incubators, heating pads, or small environmental controllers with minimal wiring and manual configuration.
