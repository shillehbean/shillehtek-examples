# RP2040 Zero

The RP2040 Zero is a small RP2040-based microcontroller board with presoldered headers and an onboard WS2812 RGB LED. Developers can use it to learn RP2040 I/O, build simple sensor or LED projects, and prototype small embedded applications using MicroPython or Arduino tools.
