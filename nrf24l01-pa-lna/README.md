# NRF24L01+ PA/LNA

This repo entry covers the nRF24L01+ PA/LNA 2.4GHz transceiver board with an external antenna for extended wireless range. Developers can use it to build long-range low-power links such as remote sensors, wireless controls, or microcontroller telemetry between devices.
