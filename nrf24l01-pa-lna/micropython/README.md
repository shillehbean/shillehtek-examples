# Micropython examples

- [`example.py`](./example.py) — MicroPython example for Raspberry Pi Pico that configures SPI and the nRF24L01 driver, sets power/speed for long range, sends a packed integer on the TX pipe, and runs a receive loop to unpack incoming integers.

Full wiring + setup notes: [manual](https://shillehtek.com/blogs/shillehtek-product-manuals/nrf24l01-pa-lna-wireless-transceiver-module-antenna-1100m-range-manual)
