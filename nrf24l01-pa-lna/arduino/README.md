# Arduino examples

- [`example_1.ino`](./example_1.ino) — Transmits a periodic "Hello from PA LNA!" packet from an Arduino using the RF24 library with PA/LNA enabled and 250kbps data rate.
- [`example_2.ino`](./example_2.ino) — Listens for incoming nRF24L01 packets on the specified pipe and prints received messages to Serial, configured for PA/LNA use and 250kbps data rate.
