# 1-Channel 12V Relay

This is a single-channel 12V relay board for switching 12V loads from a microcontroller or SBC. Developers can use it to build simple on/off control projects such as lights, fans, or motor controllers driven from Arduino, ESP32, Raspberry Pi, or MicroPython boards.
