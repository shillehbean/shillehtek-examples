# Arduino examples

- [`example.ino`](./example.ino) — Uses the Adafruit_INA219 Arduino library to initialize the INA219, set a 32V/2A calibration, and print bus voltage, load voltage, current (mA) and power (mW) to Serial every second.

Full wiring + setup notes: [manual](https://shillehtek.com/blogs/shillehtek-product-manuals/cjmcu-219-ina219-i2c-bi-directional-current-power-monitoring-sensor-module-with-soldering-and-foam-manual)
