# Arduino examples

- [`example_1.ino`](./example_1.ino) — Initializes a 16x2 LCD over I2C using the LiquidCrystal_I2C library and displays two lines of text.
- [`example_2.ino`](./example_2.ino) — Scans the I2C bus and prints any found device addresses (in hex) to the Serial console.
