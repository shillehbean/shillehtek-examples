# PCF8574 I2C LCD Backpack

The PCF8574 I2C LCD Backpack provides an I2C serial interface for HD44780-compatible character LCDs (commonly 16x2 or 20x4) using a PCF8574 I/O expander. Developers can use it to add two-wire text displays for status panels, menus, or sensor readouts on Arduino, Raspberry Pi, or MicroPython projects.
