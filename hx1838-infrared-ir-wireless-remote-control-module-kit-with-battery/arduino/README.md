# Arduino examples

- [`example_1.ino`](./example_1.ino) — Reads incoming IR signals with the IRremote library and prints the decoded command hex code to Serial.
- [`example_2.ino`](./example_2.ino) — Watches decoded IR button codes and switches an LED on or off based on specific hex values.
