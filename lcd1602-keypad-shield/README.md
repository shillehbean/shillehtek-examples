# LCD1602 Keypad Shield

The LCD1602 Keypad Shield is a 16x2 character HD44780-compatible LCD with an integrated five-button keypad that mounts on Arduino-compatible boards. Developers use it to build simple user interfaces, menus, and input-driven projects (status displays, menu selectors, or small controllers) by reading the keypad via the A0 resistor ladder and driving the LCD with the LiquidCrystal library.
