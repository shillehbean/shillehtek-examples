# Arduino examples

- [`example_1.ino`](./example_1.ino) — Initializes the 16x2 LCD and prints a two-line 'hello world' message prompting the user to press a button.
- [`example_2.ino`](./example_2.ino) — Reads the shield's five-button keypad via the A0 analog resistor ladder and displays which button is currently pressed on the LCD.
- [`example_3.ino`](./example_3.ino) — Provides a simple menu skeleton that lets the user move a cursor with UP/DOWN and trigger an action with SELECT using the keypad read from A0.
