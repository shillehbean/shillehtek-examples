# Arduino examples

- [`example_1.ino`](./example_1.ino) — Initializes the ILI9341 using the Adafruit_ILI9341 and Adafruit_GFX libraries, sets rotation, clears the screen, and prints multi-colored text ('Hello, ILI9341!').
- [`example_2.ino`](./example_2.ino) — Reads touch input from an XPT2046 resistive touch controller and prints the raw x, y, z touch coordinates to Serial when the screen is touched.
