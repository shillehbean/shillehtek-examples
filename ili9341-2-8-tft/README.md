# 2.8in ILI9341 TFT

The 2.8in ILI9341 TFT is a 240×320 SPI color display with an optional resistive touch panel. These examples show basic usage on Arduino, ESP32, and MicroPython boards so you can build small GUIs, status displays, or touch-driven interfaces.
