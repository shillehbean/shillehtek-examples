# SG92R Servo

The SG92R is a 9g micro servo with carbon-fiber gears and about 2.5 kg·cm torque, commonly used in RC models and small robots. This repo contains simple examples showing how to drive the servo from Arduino, MicroPython (Raspberry Pi Pico), and a Raspberry Pi so you can build pan-tilt mounts, control surfaces, or small robotic joints.
