# HM-10 BLE Module

The HM-10 is a Bluetooth Low Energy (BLE) serial module based on the TI CC2541. Developers can use it to add BLE serial connectivity to microcontroller projects — for example a phone-controlled LED, wireless sensor telemetry, or to send AT commands to configure the module.
