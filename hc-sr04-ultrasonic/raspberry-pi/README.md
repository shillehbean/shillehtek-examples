# Raspberry Pi examples

- [`example.py`](./example.py) — Uses Raspberry Pi GPIO 23 (Trig) and 24 (Echo) to trigger the HC-SR04, measure the echo pulse duration, and print distance in centimeters in a loop until interrupted.
