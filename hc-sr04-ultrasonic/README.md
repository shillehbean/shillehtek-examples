# HC-SR04

The HC-SR04 is an ultrasonic distance sensor that measures distance by sending a short sound pulse and timing the echo. Developers can use it to build distance-measuring, obstacle-detection, or proximity-triggered projects with Arduino, Raspberry Pi, or MicroPython boards.
