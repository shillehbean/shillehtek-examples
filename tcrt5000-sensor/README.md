# TCRT5000 Sensor

The TCRT5000 is an infrared reflective optical sensor pair used for detecting lines and nearby surfaces. Developers can use it to build line-following robots, edge detection, or simple proximity sensors; the examples include Arduino and MicroPython usage for both analog and digital reads and a 3-sensor line follower.
