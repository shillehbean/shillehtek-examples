# MG995 Servo

The MG995 is a metal-gear, high-torque 180° hobby servo (typical 5–6V supply) used for robotics and motion control. Developers can use it for pan/tilt mounts, small robot joints, or RC steering; because it draws significant current, provide an external 5–6V supply and tie grounds. This repo contains example code for Arduino, ESP32, Raspberry Pi (pigpio), and MicroPython on a Pico.
