# Arduino examples

- [`example_1.ino`](./example_1.ino) — Continuously sweeps the MG995 from 0° to 180° and back using Arduino's Servo library on digital pin 9.
- [`example_2.ino`](./example_2.ino) — Moves the servo to 0°, 90°, and 180° (then back) while using a tightened pulse-width range (500–2500 µs) for fuller travel on pin 9.
