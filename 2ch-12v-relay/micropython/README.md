# Micropython examples

- [`example.py`](./example.py) — MicroPython example for a Pico that sets GP2 and GP3 as outputs initialized HIGH (relays OFF) and alternately drives them LOW/HIGH to switch the relays, handling KeyboardInterrupt to stop.
