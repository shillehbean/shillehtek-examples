# 2-Channel 12V Relay Module

The 2-Channel 12V Relay Module provides two electromechanical relays driven by digital IN pins with active-low logic. Developers can use it to switch higher-voltage loads (lights, motors, appliances) from microcontrollers or a Raspberry Pi; example code shows how to initialize the relays in the OFF state and cycle each relay on and off.
