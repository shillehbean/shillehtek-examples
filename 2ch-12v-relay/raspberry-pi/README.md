# Raspberry Pi examples

- [`example.py`](./example.py) — Raspberry Pi Python script using RPi.GPIO to set BCM pins 17 and 27 HIGH to start OFF and then alternately toggle them LOW/HIGH to switch the relays, with clean shutdown on Ctrl‑C.
