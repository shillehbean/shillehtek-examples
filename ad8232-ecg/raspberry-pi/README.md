# Raspberry Pi examples

- [`example.py`](./example.py) — Uses an MCP3008 over SPI to read the AD8232 ECG output on a Raspberry Pi, checks LO+ and LO- GPIOs for leads-off, and prints samples at ~200 Hz.

Full wiring + setup notes: [manual](https://shillehtek.com/blogs/shillehtek-product-manuals/ad8232-ecg-module-heart-rate-ecg-monitoring-sensor-kit-manual)
