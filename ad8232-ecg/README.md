# AD8232 ECG Module

The AD8232 ECG is an analog front-end for ECG and heart-rate monitoring that provides a single-ended ECG output plus two leads-off detection pins. Developers can use it to build simple heart-rate monitors, real-time ECG plotting, or data-logging projects with Arduino, Raspberry Pi (using an external ADC), ESP32, or RP2040 boards.
