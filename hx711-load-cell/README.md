# HX711 Amplifier

The HX711 Amplifier is a 24-bit ADC load-cell amplifier used to read strain gauges and weigh scales. Developers can use it to build digital scales, force/pressure sensors, and other weight-measurement projects with Arduino, Raspberry Pi, or MicroPython-capable boards.
