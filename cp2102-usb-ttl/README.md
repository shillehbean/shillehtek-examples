# CP2102 USB‑TTL

The CP2102 USB‑TTL is a USB-to-TTL (UART) serial converter for connecting microcontrollers and other TTL serial devices to a computer. Developers use it to open serial consoles, capture debug output, send/receive data, and assist with firmware flashing and board debugging.
