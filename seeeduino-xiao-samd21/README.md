# Seeeduino XIAO SAMD21

The Seeeduino XIAO SAMD21 is a very small Arduino-compatible development board based on the Microchip SAMD21 MCU with native USB, a 10-bit DAC, and capacitive touch-capable pins. Developers can use it to build compact USB HID devices, DAC audio/signal generators, or simple capacitive touch sensors.
