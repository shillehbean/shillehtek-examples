# Arduino examples

- [`example_1.ino`](./example_1.ino) — Blinks the board's built-in LED (which is active LOW) at a 1 Hz rate.
- [`example_2.ino`](./example_2.ino) — Outputs a 100-sample 10-bit sine wave on the DAC pin (A0/D0) using a lookup table and timed writes to produce an analog waveform.
- [`example_3.ino`](./example_3.ino) — Reads a button on D3 and sends the string "Hello from XIAO!" plus Enter over the native USB HID keyboard when the button is pressed.
