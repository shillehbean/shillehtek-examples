# ESP32-S3 DevKitC-1

The ESP32-S3 DevKitC-1 is a development board based on the ESP32-S3 microcontroller with onboard peripherals such as a WS2812 RGB LED, Wi‑Fi, and capacitive touch-capable pins. These example sketches show how to control the onboard NeoPixel, connect to Wi‑Fi, scan I2C devices, and read touch input to help you get started building networked sensors and interactive prototypes.
