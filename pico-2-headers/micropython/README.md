# Micropython examples

- [`example_1.py`](./example_1.py) — Blink the onboard LED on GP25 using MicroPython with a 0.5 second on/off interval.
- [`example_2.py`](./example_2.py) — Read the RP2350 internal temperature sensor via ADC(4), convert the ADC reading to degrees Celsius, and print once per second.
