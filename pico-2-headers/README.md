# Pico 2 Headers

The Raspberry Pi Pico 2 Headers is a Raspberry Pi Pico 2 (RP2350) development board with a presoldered USB-C connector and pin headers. These examples demonstrate basic tasks — blinking the onboard LED and reading the internal temperature sensor — to help developers start building embedded projects and prototypes.
