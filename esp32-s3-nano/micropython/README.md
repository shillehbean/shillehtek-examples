# Micropython examples

- [`example_1.py`](./example_1.py) — Starts BLE advertising under the name 'ESP32-S3-Nano' using MicroPython so the board is discoverable by BLE apps.
- [`example_2.py`](./example_2.py) — Reads the analog voltage from A0 (GPIO1) via the ADC and prints the raw reading and converted voltage periodically.
