# ESP32-S3 Nano

The ESP32-S3 Nano is a compact development board based on the Espressif ESP32-S3 with Wi‑Fi, BLE, and GPIOs. This repository includes small examples you can use to build Wi‑Fi scanners, simple LED indicators, BLE broadcasters, and analog sensor interfaces.
