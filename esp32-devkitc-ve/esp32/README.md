# Esp32 examples

- [`example_1.ino`](./example_1.ino) — Checks PSRAM presence and sizes, prints heap/PSRAM stats to Serial, and attempts to allocate a 1 MB buffer in PSRAM to verify it's usable.
- [`example_2.ino`](./example_2.ino) — Connects the ESP32 to a Wi‑Fi network using the provided SSID/password and prints the board's IP address and RSSI to Serial.
