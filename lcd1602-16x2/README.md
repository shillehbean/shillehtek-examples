# LCD1602

The LCD1602 is a 16x2 character LCD (HD44780-compatible) that displays text and simple custom characters via a 4-bit parallel interface. Developers can use it to show status, sensor readings, menus, or simple UIs on microcontrollers and Raspberry Pi. This repo includes example code for Arduino, Raspberry Pi (RPLCD), and MicroPython bit-banged usage.
