# Pico 2W Presoldered Headers

The Raspberry Pi Pico 2W Presoldered Headers is a Raspberry Pi Pico 2W board (CYW43 Wi‑Fi/Bluetooth) with headers already soldered for immediate prototyping. The examples cover basic workflows: blinking the onboard LED, connecting to Wi‑Fi, and making HTTP requests using MicroPython and the Arduino core. Use these snippets to start building connected sensors, IoT prototypes, or networked controllers.
