# Micropython examples

- [`example_1.py`](./example_1.py) — Blink the onboard LED on the Pico 2W using MicroPython, using the 'LED' alias routed through the CYW43 wireless chip.
- [`example_2.py`](./example_2.py) — Connect the Pico 2W to a Wi‑Fi network in MicroPython, waiting up to 15 seconds and printing the assigned IP address on success.
- [`example_3.py`](./example_3.py) — Perform a simple HTTP GET request with urequests in MicroPython and print the response status and body.
