# Arduino examples

- [`example_1.ino`](./example_1.ino) — Blinks the onboard user LED on PC13 at 2 Hz (active LOW) using the Arduino API for STM32.
- [`example_2.ino`](./example_2.ino) — Initializes USB CDC serial and prints system uptime in seconds once per second over the USB virtual COM port.
- [`example_3.ino`](./example_3.ino) — Scans the I2C bus (I2C1 on PB6=SCL, PB7=SDA) and prints any detected device addresses to the serial console.
