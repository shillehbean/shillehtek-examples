# STM32F411 Black Pill

The STM32F411 Black Pill is a small development board based on the STM32F411CEU6 MCU. These examples demonstrate common starter tasks — blinking the onboard LED, USB CDC serial output, scanning the I2C bus, and running a simple MicroPython script — to help you begin firmware and peripheral integration work.
