# Soil Moisture Sensor

A compact soil moisture (hygrometer) probe that provides an analog voltage (AO) and a digital threshold output (DO) for measuring soil water content. Developers can use it to build plant-watering controllers, data loggers, or low-power monitoring systems with Arduino, ESP32, Raspberry Pi Pico, or Raspberry Pi + ADS1115.
