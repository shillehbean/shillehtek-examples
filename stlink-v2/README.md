# ST-LINK V2

The ST-LINK V2 is a USB in-circuit debugger and programmer for STMicroelectronics STM8 and STM32 microcontrollers. Developers use it to flash firmware, run hardware debugging sessions, and recover or inspect device flash over SWD (STM32) or SWIM (STM8).
