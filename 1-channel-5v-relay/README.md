# 1-Channel 5V Relay Board

This is a 1-channel 5V relay board that lets a microcontroller or single-board computer switch an external load via a mechanical relay. Developers can use it to build automated lights, pumps, or other switched loads; the examples show active-LOW control and safe startup behavior for Arduino, ESP32, Raspberry Pi, and MicroPython targets.
