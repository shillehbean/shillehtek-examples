# WCS1700

The WCS1700 is a Hall-effect current sensor with an analog voltage output and an optional digital overcurrent alarm pin. Developers can use it to measure DC or low-frequency AC currents and build projects such as overcurrent protection, power monitoring, or battery current logging.
