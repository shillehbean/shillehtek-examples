# MAX30102 Sensor

The MAX30102 is an integrated pulse oximeter and heart-rate sensor (IR and red LEDs with photodetector) communicated over I2C. Developers can use this breakout to build finger pulse-oximeter or heart-rate monitor projects on Arduino, Raspberry Pi, or MicroPython boards.
