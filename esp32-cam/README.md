# ESP32-CAM

The ESP32-CAM is a compact development board that pairs an ESP32 microcontroller with an OV3660 camera and a microSD slot. Developers use it to build Wi‑Fi camera projects such as live streaming web servers, image capture to SD, or MicroPython-driven snapshot utilities. This repository provides quick-start examples and notes to get a camera stream and captured images working.
