# Arduino examples

- [`example_1.ino`](./example_1.ino) — Step-by-step instructions to load and run the Arduino CameraWebServer example on an AI Thinker / Kuongshun ESP32-CAM to get a live MJPEG web stream.
- [`example_2.ino`](./example_2.ino) — Arduino sketch that initializes the camera with the AI Thinker/Kuongshun pin mapping and captures a JPEG image, saving it to the SD card.
