# DS18B20

The DS18B20 is a waterproof digital 1-Wire temperature probe that reports temperature in Celsius with good accuracy and a simple single-wire bus. Developers can use it to build temperature logging, monitoring, or thermostat projects across Arduino, Raspberry Pi, and MicroPython platforms.
