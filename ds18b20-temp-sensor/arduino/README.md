# Arduino examples

- [`example_1.ino`](./example_1.ino) — Reads temperature from a single DS18B20 on Arduino using OneWire/DallasTemperature and prints Celsius and Fahrenheit to Serial every second.
- [`example_2.ino`](./example_2.ino) — Scans a shared 1-Wire bus for multiple DS18B20 probes on Arduino and prints each sensor's temperature in Celsius.
