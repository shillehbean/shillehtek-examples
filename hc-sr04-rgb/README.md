# HC‑SR04 RGB Sensor

The HC‑SR04 RGB Sensor combines an HC‑SR04 ultrasonic distance sensor with four onboard WS2812B RGB LEDs. Developers can use it to build distance-aware indicators, proximity alarms, or simple interactive projects that change LED color based on measured range.
